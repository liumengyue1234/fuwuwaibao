## 系统简介

审思明辨——智判法案双擎系统是一款面向法律从业者的智能辅助平台，融合四大核心能力：

- 💬 **对话式问答** - 自然语言咨询法律问题
- 📖 **精准法条检索** - 快速定位相关法规
- 📋 **相似案例匹配** - 分析类案裁判要旨
- 🎯 **诉讼策略推演** - 智能生成应诉方案

---

## 功能模块

### 1. 法律问答

法律问答模块提供基于 AI 的法律问题解答服务。

**功能特点**:
- 自然语言理解，支持口语化提问
- 自动关联相关法规和案例
- 多轮对话，支持追问和澄清
- 专业法律术语解析

**使用步骤**:
1. 在输入框中描述您的法律问题
2. 点击「智能分析」按钮
3. 查看 AI 的解答和相关法规案例
4. 如有需要，可继续追问

**示例问题**:
- 劳动合同解除的经济补偿金如何计算？
- 房屋买卖合同违约的赔偿标准是什么？
- 交通事故责任如何划分？

---

### 2. 法条检索

法条检索模块提供精准的法规检索服务。

**功能特点**:
- 支持关键词检索和语义检索
- 检索结果包含法规层级、发布机关等元信息
- 可查看法规详情和完整内容
- 智能关联相关司法解释

**使用步骤**:
1. 输入检索关键词
2. 选择检索方式（语义/关键词）
3. 点击「检索」按钮
4. 浏览检索结果，点击查看详情

**检索技巧**:
- 语义检索：输入法律问题或案情描述
- 关键词检索：输入具体的法律术语
- 可组合多个关键词提高精度

---

### 3. 案例匹配

案例匹配模块提供相似案例检索服务。

**功能特点**:
- 基于案情的语义相似度匹配
- 支持按法院层级筛选
- 提供案例分析和统计
- 展示完整裁判要旨

**使用步骤**:
1. 详细描述案情
2. 选择希望检索的法院层级
3. 点击「匹配案例」按钮
4. 浏览相似案例和裁判结果

**案情描述建议**:
- 说明当事人身份（原告/被告）
- 描述案件经过
- 明确争议焦点
- 提供关键时间节点

---

### 4. 诉讼策略

诉讼策略模块提供智能化的诉讼策略分析。

**功能特点**:
- 智能切换"对方律师"视角
- 自动生成质证意见
- 提供风险评估和胜诉概率
- 支持一键导出策略报告

**使用步骤**:
1. 输入案件描述
2. 选择您的角色（原告/被告）
3. 选择是否包含风险评估
4. 点击「生成策略分析」
5. 查看分析结果
6. 可选择导出到腾讯文档

**输出内容**:
- 对方视角分析
- 质证要点
- 风险评估
- 策略建议
- 完整报告

## 快速开始

### 第一次使用

1. **配置环境**（如在本地运行）
   - 安装 Python 3.9+
   - 安装依赖：`pip install -r requirements.txt`
   - 配置 `.env` 文件

2. **启动应用**
   ```bash
   streamlit run app.py
   ```

3. **开始使用**
   - 选择需要的功能模块
   - 输入相关问题或案情
   - 获取 AI 分析结果

### 最佳实践

1. **问题描述要清晰**
   - 使用准确的法律术语
   - 提供必要的背景信息
   - 明确您的具体问题

2. **善用案例匹配**
   - 详细描述案情有助于找到相似案例
   - 参考相似案例的裁判结果
   - 分析败诉原因和胜诉关键

3. **结合法条和案例**
   - 先检索相关法规
   - 再查看相似案例
   - 最后生成策略分析

## 使用技巧

### 如何获得更准确的回答

1. **法律问答**
   - 问题要具体，避免过于宽泛
   - 可以追问，补充更多信息
   - 重要结论建议核实法条原文

2. **法条检索**
   - 使用法律专业术语
   - 多个关键词组合检索
   - 注意法规的时效性

3. **案例匹配**
   - 案情描述越详细越好
   - 包含关键证据信息
   - 注意案例的地域和时间

4. **策略分析**
   - 客观描述案件事实
   - 提供对方可能的诉求
   - 包含相关证据清单

### 导出报告

1. 在「诉讼策略」模块生成分析报告
2. 点击「一键导出到腾讯文档」
3. 分享文档链接给团队成员

## 常见问题

### Q: 系统如何保护我的数据？

A: 系统采用本地部署模式，数据仅在本地处理，不会上传到第三方服务器。请妥善保管您的 API 密钥。

### Q: AI 的回答是否可靠？

A: AI 回答仅供参考，不能替代专业律师的法律意见。建议：
- 将 AI 分析作为辅助参考
- 重要决策咨询专业律师
- 核实关键法条和案例

### Q: 如何提高检索精度？

A:
- 使用法律专业术语
- 组合多个关键词
- 选择合适的检索方式
- 查看多个检索结果

### Q: 为什么输出为 null？

A: 可能原因：
- API 调用超时
- 输入内容过长
- 网络连接问题

**解决方案**：
- 缩短输入内容
- 更换网络环境
- 重试请求
